참고 사이트

본 프로젝트를 진행하는데 참고한 사이트 목록입니다.

  • 파일 다운로드(URLDownloadToFile) - Positive Mental Attitude님 티스토리 블로그
  • 시간 측정(Chrono) - TwoPark님 티스토리 블로그
  • CPU 스레드 수 정보 획득(SYSTEM_INFO) - 마이크로소프트 공식 설명서
  • Chrono 시간값을 string으로 변환 - stack overflow 질문 답변
  • strcat 입력을 위해 string을 char로 변환 - 댄코 - 댄싱코딩님 티스토리 블로그
  • 오일러의 곱셈공식으로 원주율 구하기 - 발자취님 티스토리 블로그
  • 문자열 바꾸기(CPU 이름 부분/replace) - 젠트의 프로그래밍 세상님 티스토리 블로그
  • 소수점 반올림(점수 계산 부분/toFixed) - 올라가용님 티스토리 블로그
